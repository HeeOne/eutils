Type Alias TimeLike

TimeLike: Date | string | number

表示可以代表时间的类型。

此类型别名定义了几种可以用来表示时间的值的类型。 它可以是一个原生的 JavaScript Date 对象，该对象能精确表示某个特定的时间点。 也可以是一个字符串，通常是符合特定日期时间格式的字符串，比如 ISO 8601 格式（'2024-01-01T12:00:00Z'）。 还可以是一个数字，这个数字一般是 Unix 时间戳（从 1970 年 1 月 1 日 00:00:00 UTC 开始到指定时间点所经过的毫秒数）。
