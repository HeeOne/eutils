# Get Started

## Install & Usage

### browser

```js
<script scr="./dist/umd/@heyee/eutils.min.js"></script>
<script>
$eutils.formatNumberThousands(34566)
// console: 34,566
</script>
```

### Package

```shell
npm install @heyee/eutils
```

代码引入

```js
import { formatNumberThousands } from '@heyee/eutils'

formatNumberThousands(34566)
// console: 34,566
```
